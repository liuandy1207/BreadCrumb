# BreadCrumb

This is the Git repository for my APS360: Intro to Deep Learning final project. 

This project identifies and counts circuit components in a picture of a breadboard and returns a bill of materials (BoM). 

Link to the database on Roboflow: [https://app.roboflow.com/snowball/breadcrumb/browse](https://app.roboflow.com/snowball/breadcrumb/browse)
